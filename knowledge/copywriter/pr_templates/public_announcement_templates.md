# PR templates

## Product launch announcement

```markdown
# [Product or feature name]

## What changed

## Who it helps

## Why it matters

## Availability

## Quote or founder note

## Next step
```

## Internal launch note

```markdown
Subject:
Preheader:

What launched:
Who should use it:
How to try it:
What still needs review:
Owner:
```

## Fact-check fields

Before publication, verify:

- product name;
- release date;
- customer-visible claims;
- pricing or cost claims;
- model names;
- security or compliance statements;
- external statistics.
