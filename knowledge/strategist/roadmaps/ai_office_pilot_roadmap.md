# AI Office pilot roadmap

## Phase 1: Knowledge bootstrap

- Normalize source documents into Markdown.
- Fill common corporate context.
- Fill all seven agent namespaces.
- Run ingest and route-only checks.

## Phase 2: Real source connection

- Add company wiki and onboarding docs.
- Add marketing examples and brandbook.
- Add database schemas and KPI formulas.
- Add support FAQ and successful tickets.
- Add finance policies and legal entity details.

## Phase 3: Quality controls

- Track route accuracy.
- Track human correction rate.
- Track support escalation rate.
- Track `needs_review` rate for finance and analytics.

## Phase 4: Production hardening

- Implement production vector search.
- Add source freshness checks.
- Add approval workflows for customer-facing and finance outputs.
- Add audit logs for source usage.
