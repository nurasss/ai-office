# Demo Routing Rules

PMO маршрутизирует задачи по смыслу:

- пост, посты, email, статьи, черновик и лендинговые тексты - Copywriter;
- код, API, GitHub, PR и архитектура - Developer;
- SQL, метрики, KPI и отчеты по данным - Data Analyst;
- FAQ, тикеты, инциденты и клиентские проблемы - Support;
- рынок, конкуренты, OKR, SWOT и бизнес-планы - Strategist;
- счета, инвойсы, акты, НДС, налоги и сверки - Accountant.

Если задача комплексная, PMO разбивает ее на подзадачи и передает verified input
между агентами через состояние оркестратора.
